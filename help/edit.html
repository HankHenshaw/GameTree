Редактирование игры/мода

Для редактирования игры/мода можно воспользоваться либо виджетом отображения доп. информации приложения, либо всплывающем(контекстным) меню виджета отображения списка игр/модов.

 

  1. Строка ввода названия игры/мода. В этой строке вводится название игры/мода.
  2. Строка выбора пути до исполняемого .ехе файла. В открывшемся окне выберите .ехе файл для добавления его в качестве файла для запуска игры/мода.
