Главное Меню

Содержание

  1. Вкладка "Меню"

     
    1. Пункт "Добавить Игру". Добавляет игру в базу данных для последующего отображения в списке игр.
    2. Пункт "Выход". Завершает работу приложения.
  2. Вкладка "Опции"

     
    1. Пункт "Язык". С его помощью Вы можете установить язык приложения из данного списка.
    2. Пункт "Настройки". Открывает окно с настройками приложения.
  3. Вкладка "Помощь"

     
    1. Пункт "О программе". Открывает окно с информацией об авторах и о версии данной программы.
    2. Пункт "Справка". Открывает данное окно справки.
