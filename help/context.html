Контекстное меню виджета отображения игр/модов

Контекстное меню бывает 4 видов. Для вызова каждого из них необходима нажать ПКМ(правая клавиша мыши) в определенной области.

Для вызова данного меню, необходимо нажать ПКМ на пустой области виджета отображения.

Содержание

  1. Открывает окно добавления игры.

Для вызова данного меню, необходимо нажать ПКМ на букве.

Содержание

  1. Открывает окно добавления игры.
  2. Удаляет из списка букву со всеми играми и модами из под нее.

Для вызова данного меню, необходимо нажать ПКМ на игре.

Содержание

  1. Запускает выбранную игру.
  2. Открывает окно запуска игры с параметрами.
  3. Открывает окно добавления игры.
  4. Открывает окно добавления мода.
  5. Открывает окно редактирования игры.
  6. Удаляет из списка игру со всеми модами из под нее.
  7. Отрывает папку в которой лежит исполняемы .exe файл, указанный при добавлении игры.
  8. Отрывает папку игры в директории данного приложения из которой берутся изображения, аудиофайлы и текстовая информация.

Для вызова данного меню, необходимо нажать ПКМ на моде.

Содержание

  1. Запускает выбранный мод.
  2. Открывает окно запуска мода с параметрами.
  3. Открывает окно добавления игры.
  4. Открывает окно добавления мода.
  5. Открывает окно редактирования мода.
  6. Удаляет из списка мод.
  7. Отрывает папку в которой лежит исполняемы .exe файл, указанный при добавлении мода.
  8. Отрывает папку мода в директории данного приложения из которой берутся изображения, аудиофайлы и текстовая информация.

Внимание! При удалении, будут также удалены соответствующие папки в папки данного приложения со всеми файлами внутри.
